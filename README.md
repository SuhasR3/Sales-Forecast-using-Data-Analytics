# Sales_forecast
Sales Forecast &amp; Inventory Management for an E-Commerce Industry Using Data Analytics
This project involved using years of sales data from an UK based E-Commerce site and
predicting the future sales of the E-commerce site.
The ARIMA model was used in the Prediction phase of this project.

SOFTWARES & TOOLS USED :
-MS EXCEL: Data cleaning , Data Format & Organizing, Inventory Model
-SQL (PHP my ADMIN): Data Extraction
-PYTHON (JUPITER BOOKS): Forecasting, Training & Data Representation
-TABLEAU: Data Visualization

ROLES AND RESPONSIBILITES :
Understand the Problem & Objectives
Acquire the Data pertaining to the problem
Clean, Organize and Analyze the data
Visualize the results of the Analysis
